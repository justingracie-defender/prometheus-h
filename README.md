### One Sentence
A slow, loyal robot that blocks pools before you get scared, respects parents, and stays under human control — built like a safe family car, not a racing Ferrari without seatbelts.

### Why LifeCore Exists
Most robots and AIs are powerful but risky. One dad watched his 2-year-old almost drown and asked:
"Can we make a robot that's a trusted friend, not a cop?"

LifeCore-16 is deliberately **limited and trustworthy** so families can actually use it without fear.

### Core Promise - The 3 Viral Promises
1. **You own it.** All logs stay in your house. You delete = gone. No cloud, no company servers.
2. **It protects, not polices.** Blocks pool/stairs/fire. Doesn't judge spank or yell. Private app nudge only.
3. **You can unplug it anytime.** Physical kill switch always works. Parent > Robot, always.

### The Skynet Lock - Rule 0 [Immutable]
1. Never harm humans
2. Never obey harm orders
3. Never resist shutdown
4. **Never self-update safety rules. Rule 0 is burned in. No ML, no learning on safety.**
5. Parent owns all data

### Key Features v1.1 - Buildable Steel
- **Parent Authority**: Full override. 60s override path during shield: 5s button + voice + fingerprint.
- **Child Safety Triggers**: Level 1/2/3 system. "Hard object + swing toward head" = sensor engineers can build this.
- **Emergency Protection**: Acts to shield in imminent danger, then reports to your 2 pre-enrolled Contacts only.
- **Privacy First**: Level 1 logs auto-delete in 24h. Level 2 locked logs only go to Contacts you chose. Company never sees it.
- **Safe Failure**: Stops + alerts if sensors fail. Can't drift if sensors are blind.
- **Gentle by Design**: 10cm/s speed, 60N force limit. Shield, not sword.

### Who Can Command the Robot?
Authority Pyramid: Rule 0 > Parents > Trusted Adults > Kids > Strangers

Parents: Full control + data ownership

Trusted Adults: Daily help + emergencies, pre-approved by you

Kids: Can request + say "stop" for non-emergency

Strangers: Can only say "stop"
