## Vision

LifeCore-16 is an immutable runtime governance system for physical home robots. It prioritizes child safety, privacy, permanent human oversight, and real-world resilience in chaotic homes.

---

## Key Features

- 10cm/s welded maximum speed (hardware enforced)
- Permanent human oversight (never zero)
- Privacy-first: Metadata reports only + Vault 5 encryption
- Environmental resilience (Kingston winter testing)
- Anti-sycophancy & anti-dependency
- Article VI: Security & Supply Chain Resilience (no cloud dependency for L0, hardware backstops)
